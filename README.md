# WBS Supervised Machine Learning

**Python | pandas | scikit-learn | Classification | Regression | Random Forest | Model Evaluation**

## Project Overview

This repository contains my supervised machine learning exercises and practice projects completed during the WBS Data Science & Machine Learning course.

The main focus of this repository is to practise the complete supervised machine learning workflow, from data preprocessing and baseline modelling to model training, hyperparameter tuning, and model evaluation.

The exercises cover both major types of supervised learning:

- **Classification**: predicting categorical outcomes
- **Regression**: predicting continuous numerical values

Several case studies were used during the course, including projects such as the **Titanic Classification Project**, housing price prediction exercises, and regression-based datasets such as a **Diamonds / Price Prediction Project**.

The goal is not only to apply machine learning models, but also to understand how to build reliable, reproducible, and interpretable machine learning pipelines.

---

## Project Goal

The goal of this repository is to demonstrate my practical understanding of supervised machine learning using Python and scikit-learn.

The exercises aim to answer questions such as:

- How can labelled data be used to train predictive models?
- How can missing values and categorical variables be handled correctly?
- How can classification and regression models be compared?
- How can pipelines help avoid data leakage?
- Which evaluation metrics are suitable for different machine learning problems?
- How can model performance be improved using cross-validation and grid search?
- When should more advanced models such as Random Forest be used?
